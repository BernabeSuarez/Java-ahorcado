# Juego Ahorcado en JAVA

juego realizado en **JAVA** para repasar condicionales, bucles, variables, entrada por teclado, etc.
donde se deben ir ingresando letras por teclado para adivinar la palabra secreta en determinados intentos, si se terminan los intentos, perdes.

## Herramientas utilizadas

1. Clase Scanner
2. Clase String
3. Estructuras de control (Bucles for-while)
4. Estructura de control ( if-else)
5. Asignacion de variables
